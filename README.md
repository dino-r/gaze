# gaze
View images

Dependencies: 
   - SDL2 
   - SDL2\_image 
   - and SDL2\_ttf

List images you want to look at as command line arguments.

   - Use j and k to go to the next and previous image, respectively. 
   - Use mouse wheel to zoom. 
   - Click and drag to move image around.
